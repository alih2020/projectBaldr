Le Playstation VR en détail
Caractéristiques techniques






Nom du produit : PlayStation VR

Code produit : modèle CUH-ZVR2

Écran : OLED

Taille de l'écran : 5,7 pouces

























Résolution : 1920 x RGB x 1080 (960 x RGB x 1080)

Taux de rafraichissement : 120 Hz, 90 Hz

Champ de vision : environ 100 degrés

Microphone : intégré

Capteurs : accéléromètre, gyroscope

Connexion : HDMI, USB


 

Dimensions externes / Poids :

Casque VR : environ 187×185×277 mmv (largeur × hauteur × longueur, projection maximale exclue, arceau avec la longueur la plus courte). Environ 600 g (à l'exception des câbles).


Processeur : environ 150×39×114 mm (largeur × hauteur × longueur, projection maximale exclue). Environ 300 g.


Fonction du processeur :

Traitement audio 3D, écran social (mode miroir, mode séparé), mode cinéma
